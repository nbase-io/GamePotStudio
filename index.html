GamePot Studio

GamePot Studio는 게임 런처의 빌드, 버전, UI를 통합적으로 관리할 수 있도록 설계된 통합 실행 프로그램입니다. Windows와 macOS를 포함한 다양한 운영체제에서 실행 가능하며, 프로젝트 유지보수와 배포 효율성을 극대화합니다.

빌드 관리

  • 빌드 파일(zip, exe, app 등)을 업로드하고 관리
  • 운영체제(OS) / 플랫폼별로 빌드 분리 등록 가능
  • 빌드 등록 시 메타정보 입력 가능: 빌드명, 작성일, 변경 내역, 대상 플랫폼

버전 관리

  • 각 빌드에 대한 버전 번호 관리
  • 강제 업데이트 / 선택적 업데이트 여부 설정
  • 패치 서버와 연동하여 자동 배포 가능

런처 UI 관리

게임 런처의 UI 요소를 실시간으로 직관적으로 변경할 수 있습니다.

  • 로고 이미지
  • 배경 이미지
  • 컬러 테마
  • 버튼 스타일 및 텍스트
  • 뉴스 및 공지사항 영역 구성

파일 관리

여러 개의 패치 파일 또는 대용량 공유 파일을 업로드 및 다운로드할 수 있습니다.

게임 런처 기능

  • 자동 업데이트: 최신 패치를 자동으로 다운로드 및 적용
  • 무결성 체크: 클라이언트 파일 검증
  • 다운로드 속도 및 남은 시간 예측
  • 진행 상태 시각화 (진행 바 및 텍스트)
  • 설치 완료 후 게임 실행 버튼 제공
