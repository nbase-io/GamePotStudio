게임 런처 관리의
새로운 기준

GamePot Studio는 게임 런처의 빌드, 버전, UI를 통합적으로 관리할 수 있도록 설계된 통합 실행 프로그램입니다. Windows, macOS, Linux를 포함한 다양한 운영체제에서 실행 가능하며, 프로젝트 유지보수와 배포 효율성을 극대화합니다.

주요 기능

게임 런처 관리를 위한 완벽한 솔루션

빌드 관리

  • 빌드 파일(zip, exe, app 등)을 업로드하고 관리
  • 운영체제(OS) / 플랫폼별로 빌드 분리 등록 가능
  • 빌드 등록 시 메타정보 입력 가능: 빌드명, 작성일, 변경 내역, 대상 플랫폼

버전 관리

  • 각 빌드에 대한 버전 번호 관리
  • 강제 업데이트 / 선택적 업데이트 여부 설정
  • 패치 서버와 연동하여 자동 배포 가능

런처 UI 관리

게임 런처의 UI 요소를 실시간으로 직관적으로 변경할 수 있습니다.

  • 로고 이미지, 배경 이미지 및 컬러 테마
  • 버튼 스타일 및 텍스트 커스터마이징
  • 뉴스 및 공지사항 영역 구성

파일 관리

여러 개의 패치 파일 또는 대용량 공유 파일을 업로드 및 다운로드할 수 있습니다.

통계

사용자 다운로드 현황, 업데이트 성공률, 플랫폼별 사용량 등 다양한 통계 데이터를 확인할 수 있습니다. (준비 중)

게임 런처 기능

사용자에게 최적의 게임 실행 환경을 제공합니다

자동 업데이트

최신 패치를 자동으로 다운로드 및 적용합니다.

무결성 체크

클라이언트 파일의 무결성을 검증합니다.

다운로드 속도 관리

다운로드 속도 및 남은 시간을 정확히 예측합니다.

진행 상태 시각화

진행 바 및 텍스트로 설치 상태를 직관적으로 보여줍니다.

원클릭 실행

설치 완료 후 게임 실행 버튼을 제공합니다.
